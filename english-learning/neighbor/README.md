# Neighbor

> "Practice English With Real People Near You"

## What
Location-based language exchange with structure. Matches immigrants with native speakers, provides conversation prompts, rates difficulty, tracks confidence.

## Why
Social isolation compounds the language barrier. Structured real-world practice accelerates learning.

## How
- **iOS:** Nearby Interaction framework, iMessage extension for scheduling, HealthKit for social wellness

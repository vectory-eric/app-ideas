# The Daily Catch-Up

> "Your Personalized News, Deconstructed"

## What
A daily news briefing that presents 3-5 real headlines in layers: simplified summary -> full article -> deep dive on hard phrases -> listening version at adjustable speed.

## Why
News is something you want to consume anyway, so the motivation problem evaporates. Layered difficulty respects user intelligence unlike gamified apps.

## How
- **iOS:** Lock Screen widgets ("today's phrase"), background audio for commutes, Siri Shortcuts
- **Content:** AP/Reuters headlines, layered difficulty presentation
- **Audio:** Adjustable speed listening versions

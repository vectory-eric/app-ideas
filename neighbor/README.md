# Neighbor

> "Practice English With Real People Near You"

Location-based language exchange with structure. Matches immigrants wanting practice with native speakers. Provides conversation prompts, rates difficulty, tracks speaking confidence over time.

## Target User
Immigrants feeling socially isolated

## iOS Mechanics
- Nearby Interaction framework
- iMessage extension for scheduling meetups
- HealthKit integration for "social wellness" tracking

## Biggest Risk
Marketplace cold-start problem. Safety/trust concerns with meetups. Needs critical mass in specific metro areas.

## Status
Deprioritized — marketplace cold-start makes this a fundraising pitch, not a build.

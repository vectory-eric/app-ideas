# Between the Lines

> "Read What Americans Actually Mean"

## What
A reading companion app that explains tone, subtext, formality level, and cultural assumptions in any English text — not just translation, but comprehension.

## Why
Every immigrant screenshots confusing emails and texts friends asking "what does this mean?" This productizes that existing behavior with zero behavior change via Share Sheet.

## How
- **iOS:** Share Sheet extension, VisionKit OCR for screenshots, Action Button shortcut
- **AI:** LLM with system prompting for tone/subtext analysis
- **Moat:** Contextual memory — learns your workplace, kids' school, city, level over time

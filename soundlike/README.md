# SoundLike

> "Accent Coaching That Doesn't Judge"

Pronunciation and accent reduction. Record yourself reading or speaking freely. Phoneme-level feedback via CoreML — not "wrong" but "here's the mouth position shift to sound more natural." Adapts to which English you're targeting (American Midwest, Australian, British RP, etc.)

## Target User
Professionals who feel self-conscious in meetings

## iOS Mechanics
- CoreML pronunciation model
- ARKit front camera to visualize mouth/tongue position in real-time

## Biggest Risk
ELSA Speak has $27M+ funding and this exact positioning. Would need a sharp wedge — maybe accent coaching for specific jobs (medical, legal, tech standups).

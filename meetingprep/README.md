# MeetingPrep

> "Never Be Lost in a Work Meeting Again"

Paste a meeting agenda or calendar invite. The app pre-teaches likely vocabulary, common phrases for that meeting type (standup, 1:1, performance review, client call), and gives ready-made sentence starters. Post-meeting, log what you didn't understand for review.

## Target User
White-collar immigrants in tech, finance, corporate

## iOS Mechanics
- Calendar integration (EventKit) to auto-detect upcoming meetings
- Push 15-minute prep notification before meetings
- Shortcuts for post-meeting logging

## Biggest Risk
Narrow use case — might be a feature, not an app. Could be bundled into WorkSpeak.

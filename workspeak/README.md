# WorkSpeak

> "English for Your Actual Job"

Vertical-specific English training. Select your industry (healthcare, construction, restaurant, warehouse, tech) and get scenarios you'll actually face: calling in sick, asking for time off, understanding safety briefings, writing professional emails. Lessons are short, audio-first, and built around rehearsable scripts.

## Target User
Blue-collar and pink-collar immigrant workers

## Key Insight
Employers in the U.S. spend billions on ESL training. Start with one vertical (restaurant/hospitality — massive workforce, well-defined vocabulary, high turnover = constant new users), then sell B2B to chains.

## iOS Mechanics
- App Clips for employer onboarding (QR code in break room -> tailored mini-lesson, no download needed)
- Offline mode for workers without consistent wifi

## Monetization
- B2B to restaurant chains and employers
- Trojan horse distribution via App Clips bypasses App Store discovery problem

## Biggest Risk
Content creation per vertical is expensive. Validate with one vertical first.

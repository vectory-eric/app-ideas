# Civic English

> "Pass the Test, Understand the System"

Hyper-focused on English for civic life: citizenship tests, parent-teacher conferences, DMV, healthcare forms, tax documents, jury duty. Bureaucratic survival English. Includes real document templates to practice filling out.

## Target User
Immigrants navigating U.S./UK/AU government systems

## iOS Mechanics
- Document scanner (VisionKit) to scan real forms and get guided explanations
- Offline mode for use in government offices with no wifi

## Biggest Risk
Small TAM ceiling unless broadened. But very high-intent users — people will pay if it helps them pass a citizenship interview.
